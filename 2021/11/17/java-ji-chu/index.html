静态方法不能通过对象调用，只能通过类名调用。

静态方法可被子类继承。

类的int属性如果没有赋初值，则在创建类对象时，会被赋值为0。
JPA可能就是根据对象的id是否为0来判断表中是否存在该对象的记录，从而决定save方法是插入还是修改操作。
类的String类型的属性不会再创建对象时被赋默认值，为null。
类的对象类型属性，创建时不会赋予默认值，为null。
